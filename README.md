# XMiner

NVIDIA、AMD显卡的`ETH`, `RVN`, `BEAM`, `CFX`, `ZIL`, `ERGO`, `AE`挖矿软件。

## 特色

* 自动解决部分矿池被封禁的问题
* 人性化的超频功能，可以 根据显卡类型定义超频参数 或者 自定义手动超频
* 支持WEB端直接访问，不必蹲守在电脑前进行操作，通过直接访问您机器的地址您就可以使用其他终端（例如您的手机）直接进行 开始、停止 和 超频 等操作
